# Compiler-html-css-js
This project is a simple **Online Code Compiler (Frontend)** built using HTML, CSS, and JavaScript.   It allows users to write code in separate editors for HTML, CSS, and JavaScript and run it instantly to see the output in real time.
---

## 🚀 Features
- Live code editing (HTML, CSS, JavaScript)
- Real-time output preview
- Run and clear functionality
- Clean and responsive UI
- Theme toggle (if added)
- Save code feature (if added)

---

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript (DOM Manipulation)
